## Installation ##
Ontodia is a seperate repository on github, you can find it [here](https://github.com/metaphacts/ontodia). It is also part of the [Metaphacts](https://metaphacts.com/) offering, where you can get an up to date licensed version. However the version [here](https://github.com/metaphacts/ontodia) is still usable. 

The install is simple: clone the repo, then follow the [build and run instructions](https://github.com/metaphacts/ontodia/wiki/Build-and-run).
